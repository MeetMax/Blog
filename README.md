# 博客前台

> 本项目是一套基于Vue2的个人博客项目。


## 项目搭建

``` bash
# 将项目克隆到本地
git clone https://github.com/MeetMax/Blog-Frontend.git

# 进入项目目录
cd Blog-Frontend

#安装项目依赖
npm install

# 开启开发环境服务
npm run dev
```
>注意：本项目需要配合服务端使用，关于服务端请戳 [Blog-API](https://github.com/MeetMax/Blog-API)
